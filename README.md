# アニメGIF並列メーカー

複数のアニメGIFをグリッド状に並べ、1つのアニメGIFとして出力するブラウザアプリです。

## ファイル構成

- `index.html`
- `app.js`
- `gif.worker.js`

## 使い方

1. ブラウザで `index.html` を開く（推奨: ローカルサーバー経由）
2. `GIFファイル（複数選択）` で素材GIFを読み込む
3. 基本設定の `列数` `間隔` `透明背景` を調整
4. レイアウトプレビュー上で画像をドラッグして位置微調整、別画像の上へドロップで並び順入れ替え
5. 必要に応じて `詳細オプション` を開き、背景色 / 固定キャンバスサイズ / 出力FPS / プリセットを設定
6. 設定を変えるとプレビューは自動更新（`プレビュー更新` は手動再描画用）
5. `GIFを生成` を押して、表示されたリンクから `combined.gif` をダウンロード

## 補足

- 生成処理はブラウザ内で完結します。
- 画質は最高品質で固定されています。
- `file://` ではブラウザのWorker制限で生成に失敗することがあります。可能なら `http://localhost` で実行してください。
- 例: `npx serve .` または `python -m http.server 8000` を `d:\アニメGIF並列` で実行
- 入力GIFの再生時間が異なる場合、短いGIFはループ再生されます。
- 生成中は進捗率と残り時間（推定）を表示します。
- 同一連続フレームは自動で間引き最適化され、出力容量を削減します。
- 解像度や総処理量が大きい場合は警告を表示します。
